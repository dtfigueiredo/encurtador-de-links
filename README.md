# EncurtaLink

Projeto para encurtar links a partir da API do Bit.Ly.

Funcionalidades:

* Gerar um link encurtado que é salvo para o localStorage

## Instalação

Instale my-project com npm

```bash
  npm install my-project
  cd my-project
```
    
## Stack utilizada

**Front-end:** NextJs, TailwindCSS
